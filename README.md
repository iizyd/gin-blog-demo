## 项目简介

基于 Go、Nestjs 和 Vue3 实现的简易博客系统，可作为 Go、Vue3、Nestjs、TS 的练习项目。

## 预览地址

开发 ing

## 技术栈

- Go
  - gin
  - gorm
  - jwt-go
  - viper
  - swag
  - lumberjack
- Vue3
  - naive-ui
  - axios
  - vue-router
  - vite
  - ts
- MySQL
